# Cypress Login Sauce Demo

Proyecto de automatización E2E usando Cypress para validar el flujo de login en Sauce Demo.

## Escenarios cubiertos
- Login exitoso con credenciales válidas
- Login fallido con credenciales inválidas

## Tecnologías
- Cypress
- JavaScript
- Node.js

## Sitio probado
https://www.saucedemo.com/

## Cómo ejecutar el proyecto

```bash
npm install
npx cypress open
